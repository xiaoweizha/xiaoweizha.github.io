# Check Specify Project Status

Display current Specify configuration and project status.

检查和显示当前Specify项目的配置状态，包括：

1. **配置文件检查**
   - .specifyrc.json 是否存在和格式正确
   - Personal Access Token 配置状态
   - Repository 配置信息

2. **依赖检查**
   - @specify/cli 是否已安装
   - 版本信息

3. **项目结构检查**
   - tokens目录状态
   - 生成文件的最后更新时间
   - 配置的parser规则

4. **连接测试**
   - 测试与Specify API的连接
   - 验证token有效性
