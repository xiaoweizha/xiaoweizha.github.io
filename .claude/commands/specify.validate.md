# Validate Design Tokens

Validate the current design tokens against the project constitution and best practices.

验证当前的设计tokens是否符合项目规范和最佳实践：

## 验证项目

### 1. Token命名规范
- 检查命名是否遵循语义化原则
- 验证层级结构是否正确
- 确认状态修饰符使用规范

### 2. 值的一致性
- 颜色对比度检查
- 间距系统的数学关系
- 字体大小的层级合理性

### 3. 完整性检查
- 必需tokens是否存在
- 组件变体是否完整
- 响应式断点配置

### 4. 可访问性验证
- 颜色对比度WCAG合规性
- 触摸目标尺寸检查
- 文本可读性评估

### 5. 技术规范
- CSS变量格式正确性
- 跨浏览器兼容性
- 性能影响评估

## 输出格式
- ✅ 通过的检查项
- ⚠️ 需要注意的问题
- ❌ 必须修复的错误
- 📊 统计信息和建议
