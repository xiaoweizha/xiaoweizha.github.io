# Setup Specify Project

Interactive setup wizard for configuring Specify in this project.

交互式设置向导，帮助配置Specify项目的各个方面：

## 设置步骤

1. **检查环境**
   - 验证Node.js和npm版本
   - 检查项目结构

2. **安装依赖**
   - 安装@specify/cli
   - 验证安装成功

3. **配置Token**
   - 引导用户获取Personal Access Token
   - 验证token有效性
   - 更新.specifyrc.json配置

4. **Repository配置**
   - 设置repository名称
   - 配置parser规则
   - 选择输出格式

5. **初始同步**
   - 执行第一次token同步
   - 验证生成的文件

6. **验证设置**
   - 测试完整的工作流程
   - 提供后续步骤建议
